##  CalculatorIMC

A web application developed with **React** to calculate the **Body Mass Index (BMI)**, allowing users to quickly and easily check their health classification.

---

##  Features

- ✅ Automatic BMI calculation
- ✅ BMI classification (Underweight, Normal weight, Overweight, Obesity)
- ✅ Simple and intuitive user interface
- ✅ Input validation
- ✅ Real-time updates

##  Getting Started

### Prerequisites
- Node.js installed
- Git installed

### Installation & Run

```bash
# Clone the repository
git clone https://github.com/fabianadorado/CalculatorIMC.git


# Enter the project folder
cd CalculatorIMC

# Install dependencies
npm install

# Start the application
npm start
